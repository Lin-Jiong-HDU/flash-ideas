## ⭐⭐⭐ 关键：必须考虑的四个视角 (Four Lens)

**这是评判的关键偏好！**

| 视角 | 要求 |
|------|------|
| **Equity & Justice Lens** | 考虑对边缘化社区（原住民、种族化群体）的影响；性别歧视；无障碍性；代际公平 |
| **Practical Lens** | 实际可行的时间表；在宏大愿景与现实约束之间取得平衡 |
| **Two-Eyed Seeing Approach** | 整合**原住民知识**与**学术科学**；基于事实和研究 |
| **Systems Approach** | 考虑跨主题影响（如太阳能板与稀有矿产开采对生物多样性的影响） |

## 💡 黑客松获胜建议

### 1. 项目设计策略
```
✓ 选择1-2个主题深入，但展示跨主题思考
✓ 明确展示"四个视角"如何融入设计
✓ 强调协作与社区参与
✓ 融入原住民知识/视角（如适用）
```

### 2. 强调的关键词
- **Collective Action** (集体行动)
- **Equity & Justice** (公平与正义)
- **Indigenous Knowledge** (原住民知识)
- **Systems Thinking** (系统思维)
- **Community Resilience** (社区韧性)
- **Nature-Based Solutions** (基于自然的解决方案)

### 3. 避免
- ✗ 纯技术解决方案没有社会考量
- ✗ 忽视边缘化群体
- ✗ 缺乏实际可行性
- ✗ 单一视角（缺乏系统思考）
