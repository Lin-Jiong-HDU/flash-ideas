# Flash Ideas: 核心项目提案备选

## 提案一：Democratizing the Doomsday Radio (末日无线电平民化)

**所属主题：** Adaptation & Resilience (适应与韧性)

**核心目标：**
把极高门槛的硬件生存通讯协议，转化为普通人在断网、恐慌状态下也能一键盲操的求生 UI。

**痛点分析：**
无线电和 Mesh 网络在灾难中极其有效，但普通人根本不会用。在极端环境下，复杂的专业术语和操作界面会导致系统不可用。

**产品形态：**
一个极其轻量的 PWA 离线应用。

## 提案二：Urban Miner (城市矿工：AI 驱动的电子垃圾悬赏网络)

**所属主题：** Climate Technology & Infrastructure (气候技术与基建)

**核心目标：**
用 AI 解决电子垃圾回收分类和估值这“最昂贵的第一公里”，利用共享模式解决物流问题。

**痛点分析：**
回收企业最大的痛点是收不到货。大量富含稀有金属的电子垃圾闲置在普通人家中，传统上门回收成本过高。

**产品闭环：**
1. C端 (普通用户)：用手机扫一扫旧电子产品，AI 瞬间给出大宗商品市场的预估回收底价，用户可一键发布“矿包”悬赏。
2. B端 (回收员)：提供城市资源热力图，当某小区“矿包”累积到一定价值，系统向附近回收员派单，上门统一回收。
